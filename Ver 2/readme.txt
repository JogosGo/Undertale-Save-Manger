Save Manager App - Information

Thank you for choosing the Save Manager App!

Description:
The Save Manager app allows you to manage your game saves for the game "UNDERTALE." You can copy, rename, delete, and load different save files with ease.

System Requirements:

Operating System: Windows 7 or later
Python: Version 3.6 or later must be installed on your computer.
Installation Instructions:

Download the "Save_Manager_App.exe" file from the provided link.
Place the "Save_Manager_App.exe" file in a location of your choice.
Ensure that Python is installed on your computer.
Double-click the "Save_Manager_App.exe" file to run the Save Manager App.
Using the App:

The Save Manager App provides a user-friendly interface to manage your game saves.
Use the buttons on the app to copy, rename, delete, and load save files.
Important Note:

The "Load" function will replace your current UNDERTALE folder with the selected save. Proceed with caution.
